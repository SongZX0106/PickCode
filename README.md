# 取件码APP

### 介绍
完全使用AI编写的取件码管理APP
目前小区取快递时，会发送短信来告知取件码，当快递比较多的时候，容易遗漏，特此开发这款APP，收到短信后，把取件码记录到APP中，APP可以自动根据日期记录当天的代取件数量，取完一个后点击勾选框，代取件数量就会相应减少

### 软件架构
uni-app、vue3
